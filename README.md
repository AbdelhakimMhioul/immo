# IMMO

Immo is a simple tool for image generation using the OpenAI API.

## Installation

```bash
pnpm install
```

## Usage

- Create a .env file
- Add your OPENAI_API_KEY to the .env file
- Run the following command

```bash
pnpm dev
```
